🐧 About Custom T2 Linux Build

Community Linux distribution built from the T2 Linux framework

Custom T2 Linux Build by Orlovsky Consulting GbR

Custom T2 Linux Build is a community-maintained customized distribution based on the official T2 Linux Software Construction Kit.

⚠️ Important Notice

We are not the official T2 Linux developers.

This project creates custom builds based on the official T2 Linux source repository.

Official T2 Linux project: https://t2linux.com/

Project Goals

  • Custom T2 Linux configurations
  • Reproducible Linux builds
  • X86_64 AMD CPU target platform
  • Learning and experimenting with Linux from source
  • Server and minimal system templates
